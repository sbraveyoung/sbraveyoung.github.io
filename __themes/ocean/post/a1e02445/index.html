说说付费软件

我们这一代，很多人很小就开始接触电脑，接触互联网了。像我，小学五年级时就被人带去网吧并申请了互联网上的第一个账号——QQ 号，从此就打开了新世界的一扇大门。

在近十年的网络遨游中，我一直学习和见证了很多事物的荣辱兴衰，比如 PC 时代阿里巴巴的崛起，微信借着移动互联网的浪潮登上王者宝座，小米站在风口起飞等等，当然也有曾经的盛大游戏和四大门户网站的变迁。我使用着微软的 Windows 和 office，在 chrome 上深度优先遍历了许许多多的网页，通过知乎发现了更大的世界等等，这些都是连接人与网络的基石，为我们生活的方方面面带来了无尽的方便。当然，由于那堵看不见的墙的存在，很多优秀的软件并无法很好地体验到，不免遗憾。

然而在这十年的时间里，我年少无知，把大把时间花在了网络游戏上，就像吸食鸦片一样沉迷其中不可自拔，一度导致了学习成绩的下滑。这种情况直到我考上了大学，进入了计算机专业才有所改观。但这并不能阻挡我对那些优秀软件的溢美之词。本来是揣着玩游戏的心报了这个专业，一学习，才发现完全不是这样，而是正儿八经的编程艺术。学了计算机，意味着你将不得不减少大量的娱乐时间来弥补你与天才们的差距——他们往往在高中、初中甚至小学你还在玩泥巴的时候就已经步入了编程的艺术殿堂，现在已经难以望其项背。但我还是乐此不疲的去追赶天才们的脚步，以期将两个筋斗云的差距缩短到一个。

从大一到现在的这段时间里，我才明白开发一个软件有多么难。软件开发要顾及方方面面，但只要有一个细节没做好或者撑不住就会前功尽弃，所有的努力都化为乌有。而且就算产品做成了，还要时刻提防各种意外的出现，比如机房光纤被挖掘机挖断了，或者一些人为的问题，比如技术负责人跳槽临走时顺便删个库什么的。之前 gitlab 的删库事件和亚马逊的 S3 服务大范围瘫痪也是很好的佐证。

S3 全称 simple storge service，是由亚马逊开发的数据存储服务，互联网上有将近一半的公司都在使用，亚马逊应该是有足够重视的，但结果是连亚马逊这么大的公司都会出故障导致不得不重启服务，这足以说明开发一个合格的软件是有多困难！君不见，当你为安装可盗版 Windows、盗版 office、盗版 VMware 等软件而沾沾自喜时，无数程序员为了做出这些激动人心的作品而加班加点，通宵达旦，白了头发，虚了身体而不得不去看《颈椎病康复指南》。当然，我相信这些国外的公司有让你破解不了的方法，区别只是做与不做而已。

所以朋友们，软件和许许多多的实物一样，都包含着很多人的心血，他们理应获得对应的报酬，我们也有理由为优秀的产品买单。更何况我们还是程序员呢，程序员的苦只有程序员知道，自己人何苦为难自己人呢？否则放任国内这股盗版横行的风气生长的话，极端一点，恐怕很多开发者将会失业。要真到那个时候，只怕花费再多的钱也无力回天了。

我一直在用某款笔记软件，就像「果粉」一样是它的忠诚信仰者，并安利给了一些朋友。今年元旦开始这款软件开始收会员费，我并不意外。前几天有位朋友找我说这么一个软件竟然要收费？爷我还不伺候了，祝愿它早日倒闭。所以我今天写下这篇文章，只为了改变某些人心中的「软件就该免费」的根深蒂固的思想，哪怕只是一点点，那就再好不过了。
